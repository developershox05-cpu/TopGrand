Yuklanmoqda / Loading...
⚠️ Xatolik aniqlandi / Deployment Warning

Uzbek: Veb-sayt xom (uncompiled) rejimda yuklandi yoki brauzerda xatolik yuz berdi. Iltimos, quyidagi qadamlarni bajaring:

1. Cloudflare Pages portalida "Build output directory"ni "dist" deb sozlang. 2. "Build command" bo'limiga "npm run build" deb yozing. 3. Agar drag-and-drop orqali ZIP yuklayotgan bo'lsangiz, butun loyihani emas, faqat loyiha yaratilganidan keyingi "dist" papkasining ichidagilarini ZIP qilib yuklang! Dilshodbek, shunda sayt xatosiz bir urinishda ochiladi.

English: The web app loaded in raw mode or a runtime error occurred. Please follow these steps:

1. Set "Build command" to: npm run build 2. Set "Publish directory" / "Build output directory" to: dist 3. If using drag-and-drop, upload ONLY the contents of the compiled "dist" folder, NOT the root project folder!
